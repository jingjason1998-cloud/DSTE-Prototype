## 二、核心竞品详细分析

### 2.1 国际 EPM 套件

#### SAP Strategy Management

| 维度 | 详情 |
|------|------|
| **定位** | 大型企业战略绩效管理套件 |
| **核心功能** | 战略地图、BSC、KPI库、战略举措跟踪、风险合规 |
| **方法论** | 支持BSC，但不支持DSTE |
| **经营分析会** | ❌ 无专门功能，只有标准报表 |
| **AI能力** | ⚠️ SAP AI基础能力，无战略专用AI |
| **BI集成** | ✅ 深度集成SAP Analytics Cloud |
| **中国市场** | ⚠️ 本地化一般，实施成本高 |
| **定价** | 百万级/年 |
| **目标客户** | 大型跨国企业、央企 |

**优势**: 功能全面、品牌背书、与SAP ERP深度集成  
**劣势**: 价格昂贵、实施周期长、用户体验老旧、不支持DSTE

## 四、差异化定位分析

### 4.1 竞争格局地图

```
                    战略管理深度
                         ↑
                         │
    SAP Strategy    ●    │    ● DSTE平台（我们）
    Oracle EPM      ●    │
                         │
    ─────────────────────┼────────────────────→ 执行闭环能力
                         │
    Anaplan         ●    │    ● 飞书OKR
    用友NC          ●    │
    帆软FineBI      ●    │
                         │
                         ↓
```

### 4.2 我们的核心差异化

#### 差异化 1: 唯一支持华为DSTE方法论的平台

| 对比 | 说明 |
|------|------|
| **现状** | 市场上无任何产品支持完整的DSTE方法论（SP→BP→Execute→Review） |
| **我们的优势** | 从战略制定到执行评估的完整闭环，不是工具堆砌，而是方法论驱动 |
| **客户价值** | 学习华为最佳实践，避免自己摸索 |

#### 差异化 2: 经营分析会 — 战略执行的"中央枢纽"

| 对比 | 说明 |
|------|------|
| **现状** | 所有竞品都没有专门的经营分析会功能，会议管理停留在日程/纪要层面 |
| **我们的优势** | 会前自动材料生成 → 会中数据驱动 → 会后行动项闭环，形成执行飞轮 |
| **客户价值** | 让经营分析会真正驱动战略执行，而非流于形式 |

#### 差异化 3: 业财融合 — 预算与战略同源

| 对比 | 说明 |
|------|------|
| **现状** | EPM工具重财务轻战略，OKR工具重目标轻预算 |
| **我们的优势** | 预算即目标、预算达成率即KPI、预算调整联动目标调整 |
| **客户价值** | 战略、业务、财务三位一体，避免"战略是战略、预算是预算"两张皮 |

#### 差异化 4: AI战略助手 — 从"人找数据"到"数据找人"

| 对比 | 说明 |
|------|------|
| **现状** | 竞品AI能力停留在基础查询或简单预测 |
| **我们的优势** | 主动洞察、根因分析、What-if模拟、报告自动生成、会议助手 |
| **客户价值** | 降低战略管理专业门槛，让AI成为"战略副驾驶" |

#### 差异化 5: 帆软BI原生集成

| 对比 | 说明 |
|------|------|
| **现状** | 其他战略工具需要额外集成BI，数据割裂 |
| **我们的优势** | 与帆软FineBI原生集成，战略数据与业务数据同源 |
| **客户价值** | 已有帆软BI的客户可以无缝升级，数据资产不浪费 |

## 五、目标市场定位

### 5.1 目标客户画像

| 客户类型 | 特征 | 痛点 | 我们的价值 |
|---------|------|------|-----------|
| **华为系企业** | 已学习华为管理方法 | 缺乏DSTE系统支撑 | 唯一支持DSTE的平台 |
| **快速成长型企业** | 年营收10-100亿 | 战略执行不到位 | 从战略到执行的闭环 |
| **多元化集团** | 多业务板块 | 各板块各自为政 | 统一战略管理平台 |
| **帆软BI老客户** | 已有FineBI |  BI有了但战略管理缺 | 无缝升级，数据复用 |
| **传统制造企业** | 预算管理成熟 | 战略与预算脱节 | 业财融合 |

### 5.2 竞争策略

| 策略 | 说明 |
|------|------|
| **错位竞争** | 不与SAP/Oracle正面竞争（它们重财务），不与飞书OKR竞争（它们轻战略），定位"战略执行专家" |
| **方法论壁垒** | DSTE方法论是核心壁垒，客户一旦采用，迁移成本高 |
| **生态绑定** | 与帆软BI深度绑定，共享客户基础 |
| **AI差异化** | 用AI能力弥补中小企业战略管理人才不足的痛点 |

## 七、结论

### 市场机会

1. **空白市场**: 战略管理SaaS领域尚无DSTE方法论支持的产品
2. **痛点明确**: 企业战略执行不到位是普遍痛点，经营分析会流于形式
3. **技术就绪**: AI+BI技术成熟，可以支撑智能战略管理

### 我们的定位

> **"DSTE战略管理专家 — 让战略从纸上落到地上"**

不是又一个OKR工具，不是又一个BI平台，而是**战略执行的中央操作系统**。
