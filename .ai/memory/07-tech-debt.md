## 债务格式

```
## [债务编号] 标题
- **描述**：什么问题

- **引入时间**：什么时候产生的
- **修复成本**：低 / 中 / 高
- **建议修复时机**：什么时候修最合适
```

## 当前债务

### [DEBT-001] 生产服务器目录名与项目名不一致

- **描述**：服务器上项目目录名为 `/opt/meeting-reviewer/`，但实际承载的是整个 DSTE 战略管理平台（包含驾驶舱、材料审核、业务专题、经营分析会等全部模块）。该名称源于早期仅部署「材料审核」模块时的命名，随着系统扩展已产生歧义。
- **影响**：
  - 新接手的人容易误以为这只是材料审核模块的目录
  - 备份文件名、服务名、nginx 配置都沿用该名称，品牌一致性差
- **引入时间**：项目早期（早于 v0.3.0）
- **修复成本**：中 — 需要同步修改 nginx 配置、systemd service、deploy 脚本、SSL 路径等
- **建议修复时机**：**计划维护窗口（planned maintenance）** — 选择低峰期（如周末凌晨），一次性完成目录迁移 + 验证，避免在紧急回滚后立即操作引入二次风险
- **关联配置清单**：
  - `/etc/nginx/sites-enabled/meeting-reviewer` — root、alias、ssl_certificate 路径
  - `/etc/systemd/system/meeting-reviewer.service` — WorkingDirectory、ExecStart
  - `/opt/meeting-reviewer/deploy/deploy.sh` — APP_DIR 变量
  - 备份文件命名已改为 `dste-*`（2026-06-02 完成）
