# T010: 需求管理中心 — 需求池列表页

## 目标
实现需求管理中心的首个核心页面：需求池列表页。展示所有需求的汇总列表，支持筛选、排序、分页，以及新建需求的入口。

## 验收标准
- [ ] 页面在系统管理模块下可正常访问
- [ ] 展示需求列表：编号、标题、提出人、状态、优先级、创建时间
- [ ] 支持按状态/优先级筛选
- [ ] 支持按创建时间排序
- [ ] 支持分页（默认 20 条/页）
- [ ] "+ 新建需求"按钮，点击弹出创建弹窗
- [ ] 点击需求行可查看详情（弹窗或跳转）
- [ ] 样式与现有 DSTE 平台一致（CSS 变量、主题切换）
- [ ] Playwright E2E 测试覆盖列表渲染和筛选功能
- [ ] pytest 结构测试通过

## 执行步骤

- [ ] 新建 `src/requirement-pool.html`

- [ ] 在 `src/lib/config.js` 的 `SIDEBAR_CONFIG` 添加导航项（系统管理 > 需求管理中心）

- **断点状态**：未开始
